How to use:
Development - 

Working on the login page?
There is a folder "Login Screen" I have setup the supabase database for the login system. Install the extention "Live Server"from the left panel, open the html file, right click and open with live server. This is how you can develop and test the site without it being live. Use the designs on fima to create the login page. 

Working on a role based dashboard?
In the "Pages" file there is a skeleton for the navigation, dont edit in this file. Copy it and paste it in the needed location before you start coding, this is to keep the dashboard layout consistant between all roles. You should only need to code withing the "Main" Tags.
